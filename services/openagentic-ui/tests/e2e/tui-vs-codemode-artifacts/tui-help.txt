  OpenAgentic v0.7.0  general   commands   custom-commands


  OpenAgentic understands your codebase, makes edits with your permission, and executes commands — right from your terminal.

  Shortcuts
  ! for bash mode           double tap esc to clear input        ctrl + shift + - to undo
  / for commands            shift + tab to auto-accept edits     ctrl + z to suspend
  @ for file paths          ctrl + o for verbose output          ctrl + v to paste images
  & for background          ctrl + t to toggle tasks             meta + p to switch model
  /btw for side question    backslash (\) + return (⏎) for       meta + o to toggle fast mode
                            newline                              ctrl + s to stash prompt
                                                                 ctrl + g to edit in $EDITOR


  For more help: https://OpenAgentic platform/docs/en/overview

  Esc to cancel
