Top gaps the implementation plan addresses

  • P0 G1   Empty refused-delegation slide-out cards still render (the "0 turns / 5505 tok / no steps" cards)
  • P0 G2   Per-message header missing model + cost pill (cost only shown in topbar today)
  • P0 G3   Inline artifact width should match chat bubble width (already 902px shared token, just needs verification test)
  • P0 G4   Artifact panel is transient overlay — should be persistent right-rail (480-540px), close-on-click, multi-tab
  • P1 G5   Tool cards INPUT/RESULT JSON theme (mock: .k cyan, .s green, .n yellow)
  • P1 G6   Stages strip absent (mocks 02/03/04/06: discover → query → analyze → generate → verify)
  • P1 G7   Handoff pill absent (mock 02: gpt-oss:20b → gpt-5.2 (escalation reason))
  • P1 G8   Sub-agent stats row incomplete (today: turns + tokens; mock: turns / tokens / wall-time / $cost)
  • P2 G9   Composer toolbar order mismatch (kept ToolsIndexedPill — useful — but reorder to match mock layout)
  • P2 G10   Artifact iframe theme defense (CSS reset; minor)
