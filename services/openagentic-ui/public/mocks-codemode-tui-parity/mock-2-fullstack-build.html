# URL Shortener

A small fullstack URL shortener. FastAPI + React + Vite + Tailwind + SQLite.

## Run locally

docker compose up --build
# UI:  http://localhost:5173
# API: http://localhost:8000/docs
# Admin: http://localhost:8000/api/admin/links (basic auth: admin/admin — set ADMIN_PASS!)

## Run tests

cd backend && pytest -v
cd frontend && npm test

## API

POST/api/shortencreate short link
GET/{code}302 redirect
GET/api/admin/linkslist (basic auth)
