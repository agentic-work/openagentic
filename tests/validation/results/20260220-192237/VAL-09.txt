## Agent 1: SecurityTriageAgent

> **Role:** First responder — detects, deduplicates, classifies, and creates incidents.

### Specification

| Attribute | Detail |
|-----------|--------|
| **Framework** | LangGraph `StateGraph` with conditional routing |
| **Trigger** | Prometheus alertmanager webhook or periodic poll |
| **Output** | New incident record with severity + initial context, handed to ForensicsAgent |
| **Loop Pattern** | `poll_alerts → correlate_logs → check_duplicates → classify → create_incident → handoff` |

### Tool Arsenal

| Tool | Purpose | Call Pattern |
|------|---------|:------------:|
| `prometheus_alerts` | Fetch all firing/pending alerts | Every cycle |
| `loki_search_errors` | Search error logs for correlated failures | On alert match |
| `incident_list` | Check for existing open incidents (dedup) | Before create |
| `incident_create` | Create new incident with classification | On new incident |

### State Graph

```mermaid
stateDiagram-v2
    [*] --> PollAlerts
    PollAlerts --> CorrelateLogs : alerts found
    PollAlerts --> PollAlerts : no alerts (wait 30s)
    CorrelateLogs --> CheckDuplicates
    CheckDuplicates --> ClassifySeverity : new incident
    CheckDuplicates --> PollAlerts : duplicate found
    ClassifySeverity --> CreateIncident
    CreateIncident --> HandoffToForensics
    HandoffToForensics --> PollAlerts : continue monitoring
```

### Severity Classification Logic

```python
def classify_severity(alert: dict, correlated_errors: int) -> str:
    """Triage classification heuristic."""
    if alert.get("severity") == "critical" and correlated_errors > 50:
        return "sev1"  # Total outage / critical security breach
    elif alert.get("severity") == "critical" or correlated_errors > 20:
        return "sev2"  # Major degradation
    elif alert.get("severity") == "warning" or correlated_errors > 5:
        return "sev3"  # Minor issue
    else:
        return "sev4"  # Informational
```
