# Maintainers

OpenAgentic is maintained by **Agenticwork™ LLC**.

| Maintainer | GitHub | Role |
|---|---|---|
| Agenticwork LLC | [@agentic-work](https://github.com/agentic-work) | Lead maintainer / project steward |

## Responsibilities

Maintainers review and merge pull requests, triage issues, set the roadmap,
and cut releases.

**All changes to `main` require review and approval by a maintainer.** The
`main` branch is protected; no change is merged without maintainer sign-off.
Opening a pull request does not entitle it to be merged.

## Becoming a maintainer

Contributors who demonstrate sustained, high-quality involvement may be invited
to become maintainers at the discretion of the existing maintainers.
Maintainership is granted, not claimed.
