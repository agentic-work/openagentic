--------------------------------------------------------------------------------
  TROUBLESHOOTING
--------------------------------------------------------------------------------

1. If authentication fails:
   - Check AUTH_PROVIDER is correctly set
   - Verify OAuth client ID/secret are correct
   - Ensure redirect URIs match your domain

2. If Code Mode (VS Code) fails:
   - Check MinIO is running and accessible
   - Verify bucket exists: openagentic-workspaces
   - Check code-manager logs: kubectl logs -l app.kubernetes.io/component=code-manager

3. If MCP tools aren't available:
   - Check mcp-proxy pod status
   - Verify Redis connection (needed for tool state)
   - Check mcp-proxy logs for connection errors
